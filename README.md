# Android Strings XML Translator

A Next.js web application for translating Android string resources using AI. Supports multiple languages, preserves XML formatting, and handles placeholders correctly.

## Features

- Upload `strings.xml` files via drag-and-drop
- Parse and extract string resources
- Select target languages for translation
- AI-powered translation with formatting preservation
- Preview translations in a table view
- Edit translations manually
- Export translated files as properly formatted XMLs
- Download all translations as a ZIP archive

## Tech Stack

- Next.js with TypeScript
- Material-UI components
- Tailwind CSS for styling
- XML parsing with xml2js
- File handling with JSZip
