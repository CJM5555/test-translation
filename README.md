# Android Strings XML Translator

**Translate Android `strings.xml` files effortlessly using AI.**  
A modern, web-based tool built with Next.js that supports multilingual translation and preserves XML structure.

**Live Demo:** https://android-translation.vercel.app/

---

## Features

- Drag-and-drop upload for `strings.xml` files
- AI-powered translation that preserves XML formatting
- Support for multiple target languages
- Preview translations in a table view
- Manual editing of translations
- Export translated files as valid Android XML
- Download all translations as a ZIP archive

---

## How to Use

1. Visit https://android-translation.vercel.app/
2. Upload your strings.xml file.
3. Select one or more target languages
4. Let the AI translate your strings
5. Preview and edit translations if needed
6. Export the translated XML files and download them all as a ZIP

---

## Tech Stack

- **Framework:** Next.js with TypeScript
- **UI:** Material-UI and Tailwind CSS
- **XML Parsing:** `xml2js`
- **File Handling:** `JSZip`

---

## Installation Steps for Local Setup

- Clone this project to local
- Run `npm install` to install dependencies
- Run `npm build` to build project
- Run `npm start` to run project at localhost
- Visit `http://localhost:3000/`
